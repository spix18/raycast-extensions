# Reboot to BIOS Changelog

## [Initial Version] - 2025-11-25

### Added
- Initial release of Reboot to BIOS extension
- One-click reboot to BIOS firmware settings or Advanced Boot Options
- Silent operation with no command windows or UI dialogs
- Smart fallback from BIOS firmware mode to Advanced Boot Options
- Optional confirmation dialog with user preference toggle
- Automatic admin privilege request handling
- Windows platform support
