# Reboot to BIOS Changelog

## [1.0.0] - {PR_MERGE_DATE}

### Added
- Initial release of Reboot to BIOS extension
- One-click reboot to BIOS firmware settings
- Cancel Reboot command to abort pending reboots
- Smart cancel detection (shows different message if no reboot pending)
- Configurable countdown with cancel option in toast
- Cross-command communication via marker file
- Silent operation with no command windows or UI dialogs
- Optional confirmation dialog with user preference toggle
- UEFI detection with helpful error messages
- Windows platform support
